## 남은 개선 방향

### 1. 수익률 개선 (벤치마크 대비)
- 현재 +10.35% vs Buy&Hold +65.10% — 벤치마크의 16%에 불과
- 포지션 비중 5% 고정이 원인 (보수적 리스크 관리)
- 동적 포지션 사이징 또는 복수 심볼 운용으로 자본 활용도 개선 가능

### 2. 과적합 개선
- 18/18 fold가 gap > 0.3 기준 과적합 (Train-Val Gap 평균 0.25)
- 정규화 강화, 피처 추가 축소, early stopping 등 검토

### 3. 숏 포지션 활용
- 현 모델의 매도 예측 정확도가 낮아 숏은 비수익적
- 매도 전용 모델 또는 비대칭 threshold 검토

### 4. 데이터 관리
- raw 데이터 재수집 시 processed parquet이 무효화되는 문제 발생
- processed 데이터의 버전 관리 또는 git-lfs 저장 권장
- 재수집 전 반드시 processed 데이터 백업
